# Assignments and Assessments
A curated collection of school assignments and technical coding assessments in Java, Python, C++, and JavaScript.

## Quick Start
```bash
git clone https://github.com/akileshjayakumar/assignments-and-assessments
cd assignments-and-assessments
```

## Capabilities
- Git
- Java JDK 8+
- Python 3
- C++ compiler (`g++`)

## Configuration
- No required environment variables are documented for basic usage.

## Usage
```bash
# Example: React technical assessment
cd technical-sssessments/2024-GovTech-SWE-Assessment
npm install
npm start
```

## Contributing and Testing
- Contributions are welcome through pull requests with clear, scoped changes.
- No automated test suite is currently documented for this repository.
