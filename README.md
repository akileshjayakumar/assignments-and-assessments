# Assignments and Assessments

A curated collection of school assignments and technical coding assessments in Java, Python, C++, and JavaScript.

## Quick Start

### Prerequisites
- Git
- Java JDK 8+
- Python 3
- C++ compiler (`g++`)
- Node.js 18+ and npm (for the React assessment)

### Install
```bash
git clone https://github.com/akileshjayakumar/assignments-and-assessments.git
cd assignments-and-assessments
```

### Run
```bash
# Example: React technical assessment
cd technical-sssessments/2024-GovTech-SWE-Assessment
npm install
npm start
```

## Features
- Organized by category: `school-assignments/` and `technical-sssessments/`.
- Multiple language implementations (Java, Python, C++, JavaScript).
- Per-project READMEs and licenses for focused setup and usage.

## Usage
```bash
# Example: Java assignment
cd school-assignments/rainbow-table/code
javac Rainbow.java
java Rainbow
```

For assignment-specific commands, open the README inside each project folder.

## Configuration
- No shared environment variables are required at repository level.
- Some projects require local input files in their own directories (see each project README).
